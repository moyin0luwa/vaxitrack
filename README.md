# 🩺 VaxiTrack – Vaccination Tracking System

**VaxiTrack** is a full-stack vaccination tracking system that allows administrators to manage vaccines and users, and allows users to track their vaccination records, dosing schedules, and related health data.

The project is divided into two main parts:
- **Backend**: Node.js + Express + MongoDB  
- **Frontend**: React + Vite + Tailwind CSS  

---

## 📌 Features

### **User Features**
- Register and log in with phone number & password.
- View personal profile and vaccination records.
- Receive details on when next vaccination is due.
- Track upcoming doses with scheduled reminders.
- Secure authentication.

### **Admin Features**
- Manage vaccines (create, update, delete).
- Dispensing to users.
- View vaccination statistics.

---

## 🛠️ Technologies Used

### **Backend**
- **Node.js** – Server-side runtime.
- **Express.js** – Web framework for routing & middleware.
- **MongoDB** – NoSQL database for storing users, vaccines, and records.
- **Mongoose** – MongoDB object modeling.
- **dotenv** – Environment variable management.
- **cors** – Enable secure API calls from frontend.

### **Frontend**
- **React.js** – Frontend framework for building the UI.
- **React Router DOM** – Client-side routing for multiple pages.
- **Tailwind CSS** – Utility-first styling.
- **Axios** – HTTP client for API requests.
